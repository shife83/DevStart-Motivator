# DevStart-Motivator 🚀

A lightweight Windows Batch (`.bat`) script designed to automate a developer's morning routine, clear desktop clutter, and kickstart the day with a dose of coding motivation.

With a single click, this script displays a random motivational tech quote and launches all your essential coding tools simultaneously.

---

## ✨ Features

* **Daily Motivation:** Randomly selects and displays an inspiring programming quote every time you run it.
* **Workspace Automation:** Automatically opens:
    * Visual Studio Code.
    * Git Bash (or your preferred terminal).
    * Your GitHub dashboard (or any URL) in your default browser.
* **Lightweight & Fast:** Runs instantly natively on Windows without needing any extra dependencies or frameworks.
